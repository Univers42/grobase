Grobase · GitHub connect relay

This page is the public relay for connecting a GitHub organization to a Grobase org. It holds only an HMAC relay secret — never the GitHub App private key, and never a GitHub token.

Start a connect from your terminal:

42ctl org github connect <org>

That command prints an install URL pointing here; installing the App returns you via /api/callback, which signs and forwards the installation to the Grobase control plane. Your terminal finishes the connection by polling for the result.
