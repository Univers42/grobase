5 · Network perimeter (m140)

Defense in depth, outer → inner. The only container binding a public port is the WAF; Kong sits behind it on 127.0.0.1. A compromised edge container cannot open a raw socket to an engine — it has no bridge to the data plane.

Cloudflare front-door
ROADMAP recipe (hosted): proxied DNS, managed WAF, rate-limit, Turnstile, full-strict TLS + authenticated origin-pull mTLS — a 2nd perimeter on top of the in-stack WAF.
OWASP ModSecurity v3 + CRS v4 (WAF)
Sole public listener, blocking. SQLi 942100 · XSS 941100 · traversal 930100 · anomaly 949110 → 403. Benign passes; Kong-direct=404 negative control.
Kong API gateway
Bound 127.0.0.1 only (dev) — behind the WAF. Edge IP-restrict on /admin/v1/* (private CIDRs). Per-tenant IP allowlist (m106).
Application planes — edge
net-edge: waf, kong, gotrue, postgrest, realtime, studio. WAN ingress.
Control plane
net-control (internal:true, no WAN egress): tenant-control, orchestrator, permission-engine, schema-service, vault.
Data plane — engines
net-data (internal:true): postgres, mysql, mongo, redis, minio … Reachable ONLY via the dual-attached front-door routers, where owner-scope/RLS ABAC is enforced per request. edge ↛ data REFUSED.
